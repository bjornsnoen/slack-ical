Slack/iCal integration

Hi, this tool gives you a user token to link Mac iCal with slack.

You might need this if you have

  • Zoom meetings
  • Outlook calendar
  • Microsoft Teams meetings
  • Google Calendar meetings
... all synced to your iCal, but only two of those have slack integrations that update your status accordingly. This takes the different approach of integrating your local iCal with slack, so it doesn't matter if it's a personal event or what, if it's in your calendar it will update your slack status.

The only permission we require is user.profile:write because that's what's necessary in order to update your slack status. This is necessary as Slack has deprecated the old style personal API tokens and made it impossible to generate new ones. This app literally just asks for your authorization to generate the token you will get on the next page. We don't store it anywhere. The source code is freely available on my github.

Instructions for what to do with the token are also on the next page, but spoiler alert it includes using homebrew to install ical-buddy and slack-cli so if you don't have homebrew already you can start by installing that right away and leave it running while we get that user token.
